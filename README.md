# Shreyash Agare - Portfolio Website

A modern, responsive portfolio website built with HTML, CSS (Tailwind), and JavaScript. This website showcases Shreyash's design work and professional experience.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Interactive Elements**: Hover effects, smooth scrolling, and mobile menu
- **Performance Optimized**: Fast loading with optimized images and code
- **Accessibility**: Proper semantic HTML and keyboard navigation support

## Sections

1. **Navigation**: Fixed header with smooth scrolling navigation
2. **Hero Section**: Introduction with animated text and profile image
3. **Projects**: Showcase of design projects with categories and descriptions
4. **About**: Personal story and background information
5. **Toolbar**: Skills and tools used in design work
6. **Connect**: Contact information and call-to-action

## Technologies Used

- **HTML5**: Semantic markup structure
- **Tailwind CSS**: Utility-first CSS framework for styling
- **JavaScript**: Interactive functionality and animations
- **Google Fonts**: Inter, Lato, and custom Chubbo font family

## Setup Instructions

1. **Clone or Download**: Get the project files
2. **Open in Browser**: Simply open `index.html` in any modern web browser
3. **No Build Process**: This is a static website that doesn't require any build tools

## Customization

### Colors
The color scheme is defined in the Tailwind config within the HTML file:
- Primary: `#ef8354` (Orange)
- Secondary: `#2d3142` (Dark Blue)
- Accent: `#f4f0eb` (Light Cream)
- Cream: `#f8f6f3` (Off White)
- Gray: `#909096` (Medium Gray)

### Content
To customize the content:
1. Edit the text in `index.html`
2. Replace images with your own (update the file paths)
3. Modify the project details in the Projects section
4. Update contact information in the Connect section

### Styling
- Modify Tailwind classes in the HTML
- Add custom CSS in the `<style>` tag or create a separate CSS file
- Update the Tailwind config for custom colors, fonts, or spacing
